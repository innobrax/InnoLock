# Tarefas Codex - Frontend Web

## Objetivo

Criar interface web para administração e monitoramento do Inno Lock.

## Módulos

- login;
- dashboard;
- empresas;
- usuários;
- veículos;
- dispositivos;
- locais autorizados;
- viagens;
- autorização excepcional;
- eventos;
- auditoria.

## Tarefa 1 - Definir stack

```text
Sugira uma stack frontend para o InnoLock considerando painel administrativo moderno, autenticação, mapas, tabelas, filtros e deploy em servidor Linux. Não crie código ainda.
```

## Tarefa 2 - Wireframe textual

```text
Crie um wireframe textual detalhado para o frontend web do InnoLock com telas, menus, componentes e fluxos principais.
```

## Tarefa 3 - Dashboard

```text
Crie estrutura inicial do dashboard com indicadores de veículos ativos, dispositivos online, viagens em andamento, aberturas do dia, exceções pendentes e alertas.
```

## Tarefa 4 - Tela de autorização excepcional

```text
Crie tela para a central aprovar abertura excepcional. Deve mostrar veículo, motorista, localização atual, motivo, histórico recente e botões Aprovar/Negar.
```
